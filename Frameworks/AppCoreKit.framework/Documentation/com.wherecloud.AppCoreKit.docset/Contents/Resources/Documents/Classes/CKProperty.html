CKProperty Class Reference

Inherits from NSObject
Conforms to NSCopying
Declared in CKProperty.h

Overview

CKProperty is a wrapper around keyvalue coding. It allow to set/get value for an object/dictionary keypath and manage some introspection calls to provides an easy interface to access attributes and class property descriptors. Moreover it provides some methods to work with NSArray properties (insertObjects/removeObjectsAtIndexes/removeAllObjects/count).
