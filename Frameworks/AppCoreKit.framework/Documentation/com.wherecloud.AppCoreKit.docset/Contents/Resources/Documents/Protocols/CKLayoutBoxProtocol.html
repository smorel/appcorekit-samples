invalidatedLayoutBlock

This block gets called when parts of the layout has been invalidated. You can implement this block to get notified that the global size of the layouted UIView could change (For example a tableViewCell’s contentView). This block must be set on the root view contaning layoutBoxes. The subViews of subBoxes will not call this for performance reasons.

@property (nonatomic, copy) CKLayoutBoxInvalidatedBlock invalidatedLayoutBlock

Discussion
