beginBindingsContextWithScope:

It often happend that we want to manage several bindings context per object instance cause all the bindings do not have the same life span. Or we want to manage bindings in a class and other bindings in an inherited class. Scope is here to allow multiple set of binding per instance.

- (void)beginBindingsContextWithScope:(NSString *)scope

Discussion
