UIViewController(AppCoreKit) Category Reference

Declared in CKViewController.h

Overview

CKViewController is the base class providing helpers to build a view controller and manage customization with stylesheet. It also provides convenient blocks to define a view controller inline instead of subclassing it. You can define some elements for navigation like left/right bar buttons and the supported interface orientations without having to subclass it.
