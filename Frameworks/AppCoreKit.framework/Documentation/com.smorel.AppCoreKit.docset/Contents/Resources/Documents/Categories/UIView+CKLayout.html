inflateViewFromStyleWithId:

This method will lookup for a style with the specified id in the global stylesheet scope. This style requiers a “@class” key to be defined and the class must be a subclass of UIView. You can define a layout and any of the view hierarchy property and style in this scope. The view that gets returned will get the subview hierarchy style applied to it.
