CKSnapshotController Class Reference

Inherits from NSObject
Declared in CKSnapshotController.h

Overview

A class for easily define view for offscreen snapshots This is particularily usefull if you want to create an image from a complex view with logic for setting it as a passbook strip or background image as weel as for custom transitions between view controllers.

snapshotViewController:size:completion:

Call this method to snapshot this view controller’s view offscreen.

- (void)snapshotViewController:(UIViewController *)viewController size:(CGSize)size completion:(void ( ^ ) ( UIImage *image ))completion

Discussion

Call this method to snapshot this view controller’s view offscreen.

Sequence : - viewDidLoad - viewWillAppear - viewDidAppear - UIViewController(CKSnapshotController) : if the specified viewController has requiersDelayBeforeSnapshot = YES, the snapshot will wait until readyForSnapshot is called on the view controller

 - When readyForSnapshot() is called:
 - Snapshots the view
 - viewWillDisappear
 - viewDidDisappear
 - completion(snapshot)
