cancelOnEnterBackground

Setting this value to YES will cancel the action sheet when the app enters in background. Default value is NO. This parameter has no effect if no cancel button has been added to the action sheet.

@property (nonatomic) BOOL cancelOnEnterBackground

Discussion

Setting this value to YES will cancel the action sheet when the app enters in background. Default value is NO. This parameter has no effect if no cancel button has been added to the action sheet.
