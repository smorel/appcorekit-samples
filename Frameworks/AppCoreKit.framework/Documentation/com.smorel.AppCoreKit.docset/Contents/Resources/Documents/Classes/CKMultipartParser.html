CKMultipartParser Class Reference

Inherits from NSObject
Declared in CKMultipartParser.h

Overview

Parse a MIME multipart message (eg. form-data), but does not parse the individual chunks more than necessary.

   BOOL nextChunk = [parser skipPreamble];
   while (nextChunk) {
      NSData *headerChunk = [parser readHeadersData];
      NSData *bodyChunk = [parser readBodyData];
      nextChunk = [parser readBoundary];
   }
