CKResourceManager Class Reference

Inherits from NSObject
Declared in CKResourceManager.h

Overview

This class is a weak bridge to the ResourceManager framework available at https://github.com/wherecloud/ResourceManager If the resource manager is used into your app, the calls to CKResouceManager will be redirected to the instanciated ResourceManager. Else, the CKResourceManager will redirect the calls to the NSBundle mainBundle.
