Overview

The resource Manager will manage the sync between your application resources and a dropbox repository and/or a directory on your computer’s file system. Using the dropbox sync allow to synchronize resource modifications to your device. Using a local repository allow to synchronize resource modifications in the simulator only.

Resources from dropbox will get store in your application’s cache directory. When changes occurs on dropbox or your local directory, The resource manager will look for the last modified file from dropbox/local file system and the applications bundle. The newer is the winner and the resource manager will fire a notification for you to reload your UI or data models using this new file.

https://www.dropbox.com/developers/core/sdks/ios
